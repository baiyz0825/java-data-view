# 项目说明

使用JavaWeb开发的数据库显示界面的课程设计

# 项目采用技术

## 后端

1. Junit测试
2. JDK17 进行开发
3. 采用Jar包
    1. Druid数据库连接池
    2. QueryRunner
4. MySql数据库进行存储
5. 使用Tomcat服务部署

## 前端

1. jsp
2. js
3. css
