# 项目说明

使用JavaWeb开发的数据库显示界面的课程设计

# 项目采用技术

## 后端

1. Junit测试
2. JDK17 进行开发
3. 采用Jar包
   1. Druid数据库连接池
   2. QueryRunner
   3. commons beanUtils
   4. commons dbUtils
   5. commons fileUpload
   6. commons codec
   7. commons io
   8. java.utils.UUID
   9. Gson(google json包)
4. MySql数据库进行存储
5. 使用Tomcat服务部署
6. Json数据传输

## 前端

1. jsp
2. js
3. css
4. Ajax请求（请求Json）
